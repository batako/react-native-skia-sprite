# プライバシーポリシー

最終更新日: 2025年11月19日

RN Skia Sprite Animator（以下「本アプリ」）は Batako Studio が提供するアプリです。本アプリでは、個人を特定できる情報を収集・保存・送信しません。作成したスプライトアセットや JSON ファイルは、ユーザーが明示的に共有または書き出し操作を行わない限り、端末内にのみ保存されます。

## データ保存

- プロジェクトファイルは Expo FileSystem API を利用し、端末ローカルに保存されます。
- 本アプリには分析サービス・クラッシュレポート・第三者トラッキング SDK は含まれていません。

## サードパーティサービス

- ドキュメントピッカー、ローカライゼーション、共有機能などに Expo モジュールを使用していますが、これらが Batako Studio に個人情報を送信することはありません。

## サポート窓口

- サポートフォーム（Google フォーム: https://forms.gle/qYn7Dza4rXuSa1498 ）からご連絡いただいた内容は、回答目的のみに利用します。送信データは Google フォーム上に保存され、Google のプライバシーポリシーが適用されます。

## 変更について

- 本ポリシーの内容を変更する場合は、アプリ内および本リポジトリ上で告知します。変更後も本アプリを利用された場合は、改訂後のポリシーに同意いただいたものとみなします。
